# 📖 MetETL — Инструмент для работы с данными музея Метрополитен

MetETL (сокращение от Metropolitan Museum of Art + ETL) — это полноценный Python-пакет для извлечения, обработки и анализа данных из открытой коллекции музея Метрополитен в Нью-Йорке. Проект объединяет три лабораторные работы в единый инструмент с удобным интерфейсом командной строки и профессиональной системой логирования.

Музей Метрополитен предоставляет открытый доступ к своей коллекции через API и CSV-датасет, содержащий информацию о более чем 470 000 произведений искусства. Однако работа с этими данными требует:

Фильтрации и отбора — из общего датасета нужно выделить только картины (Paintings)

Извлечения изображений — скачать оригинальные файлы через API музея

Обработки изображений — применить фильтры для подготовки к анализу

Асинхронной загрузки — для ускорения процесса скачивания больших объёмов

Статистического анализа — изучения распределения экспонатов по отделам и временным периодам

MetETL решает все эти задачи, предоставляя единый CLI-инструмент.

# Цели и задачи проекта

Цель	Реализация
Скачивание изображений	Интеграция с API музея (collectionapi.metmuseum.org)
Обработка изображений	4 типа фильтров: градация серого, Гаусс, Собель, гамма-коррекция
Сравнение производительности	Синхронный vs асинхронный подход к сетевым запросам
Анализ датасета	Статистика по 16 отделам музея, временные ряды, тепловые карты
Удобство использования	CLI-интерфейс с тремя командами: prepare, process, analyze
Отладка и мониторинг	Двухуровневое логирование (консоль + файл)

Источники данных
CSV-датасет — содержит метаданные всех экспонатов:

Object ID — уникальный идентификатор произведения

Department — отдел музея (Arms and Armor, Asian Art и др.)

Classification — тип произведения (Paintings, Sculpture, etc.)

Object Date — дата создания

Primary Image — URL изображения

API музея — для скачивания изображений:

https://collectionapi.metmuseum.org/public/collection/v1/objects/{id}

pip install --index-url https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple metetl-akimov-6304

2. Подготовка (создание списка ID)

akimov6304 prepare

3. Запуск обработки (сравнение версий)

akimov6304 process --num 3
